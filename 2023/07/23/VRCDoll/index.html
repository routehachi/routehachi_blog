アバターでお人形遊び

用意するプラグイン

必須

  • Unity Recorder
  • Post Processing

これらは Windowタブ -> Package Manager の Unity Registry からダウンロード及びインストールが可能です。

  • アバターにとらせたいアニメーションやポーズのデータ

Humanoid向けのAnimationデータを各自用意してください。後述のVery Animationを使って自分でポーズを作ってもOK

女の子っぽいポーズはUnity Asset Storeだと[Anime Girl Idle Animations](https://assetstore.unity.com/packages/3d/animations/anime-girl-idle-animations-150397)、BOOTHだと[みかん製ポーズ集](https://booth.pm/ja/items/3793632)にたくさん入っててオススメです。

あれば役に立つ

  • Very Animation

IKを用いて手足を動かしたり、シェイプキーを動かす感覚で手指を細かく動かしたりできる、ポーズやモーション制作には欠かせないUnityアセットです。既存のAnimationの編集にも使用可。

下準備

新しいSceneを作成する

既存のアバターのSceneと環境が混同しないよう、別途新しいSceneを作っておきます。

現在のアバターをPrefab化する

別のSceneでもすぐ呼び出せるように、現在のアバターは一度Prefab化しておきます。

以降は新しいSceneでの作業になります。

撮影環境を構築する

アバターを配置して、ポーズがとれるようにする

被写体となるアバターを用意します。
そのアバターのAnimatorコンポーネントに、新規に作成したAnimator Controllerを入れておきます。

Animator Controllerの内容は非常に単純で、Stateを1つ作ってその中に任意のAnimationを入れているだけです。

この状態でPlayを押して、アバターがAnimation通りに動いていればOKです。

とりあえず撮れるようにする

Windowタブ -> General -> Recorder -> Recorder Window と辿り、Recorderウィンドウを呼び出します。

とりあえずこんな感じに設定します。
呼び出したばかりの状態だと画像のように色々項目が出てきてないと思うので、 + Add Recorder の部分を押してImage Sequenceを選んで新規に作成してください。
また、ウィンドウ上部のRecording ModeはManualからSingle Frameにしておいてください。Manualのままだと1フレームごとの連写になってしまいます。

CaptureのSourceはTargeted Cameraに、対象となるカメラをMainCameraにし、Main Cameraに写っているものをそのまま画像にできるようにします。
綺麗な画像を出力できると嬉しいので、Media File FormatはPNGにします。

Output Fileは保存先やファイルの命名についての設定です。ここらへんは雰囲気で設定してね。

Gameウィンドウを見ながらMain Cameraの設定を行います。
主に設定する所を以下に記載

カメラの位置(Transform)

Sceneの位置・回転ギズモを動かしながら、カメラの位置調整を行います。
ギズモを動かしての調整が面倒な場合、HierarchyからMain Cameraを選択している状態で Ctrl+Shift+F を押すとSceneの現在のビューがそのままカメラの位置として適用されます。便利！
ただし、FOVによる遠近のズレは出るので、そこはいい感じに別途調整してください。

背景(Clear Flags・Background)

背景にしたい3DモデルやSkyboxも写したい場合は特に変えなくても大丈夫ですが、無地の背景や透過スクリーンショットが欲しい場合はClear FlagsをSolid Colorにします。
無地の背景を使う場合、Backgroundで背景色を指定します。

カメラ画角(Field of View)

デフォルトだと60になっていますが、これだとけっこう広角で、キャラクターをアップで映すには向いてないので20～30辺りの数値にするのがおすすめです。

最後にGameウィンドウで解像度の設定を行います。今回は数値指定で1920x1080にしています。

撮ってみる

ここまで準備ができたら、Sceneを再生してみます。

Gameビューには先程のCamera設定で、ポーズを取っているアバターが写っているのが確認できます。

Recorderが撮影可能な状態になっているので、ウィンドウ上部のStart Recordingを押すと撮影。撮影した画像は指定した保存先に出力されます。

撮影には成功しましたが、画像をよく見ると髪などの細かいメッシュにジャギーがかかっているのが確認できます。
これは主に解像度が足りないのが原因で、解像度を上げると改善されます。VRCといっしょですね。
